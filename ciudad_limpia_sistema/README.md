
# Ciudad Limpia - Sistema Inteligente de Gestión de Basura

Stack:
- Backend: Laravel (PHP)
- Frontend: HTML, CSS, JS
- Base de datos: PostgreSQL

Funciones principales:
- Gestión de contenedores de basura
- Monitoreo de nivel de llenado
- Gestión de rutas de camiones
- Reportes ciudadanos
- Dashboard administrativo
